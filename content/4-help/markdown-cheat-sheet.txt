Markdown is a practical way to edit web pages.

## Basics

Formatting text:

    Normal *italic* **bold** `code`

Making a list:

    * item one
    * item two
    * item three

Making an ordered list:

    1. item one
    2. item two
    3. item three

Making a header:

    # Header 1
    ## Header 2
    ### Header 3

Making links:

    [Link to page](/help/how-to-make-website) 
    [Link to website](http://datenstrom.se)  
    https://en.wikipedia.org/wiki/Markdown

Adding an image:

    ![image](picture.jpg)

Adding emoji:

    :smile: :grinning: :angry: :heart: :coffee: :+1:

For a list of available emoji see [emoji cheat sheet](http://emoji-cheat-sheet.com).

## Shortcuts

`[image picture.jpg]` = add [images and thumbnails](https://github.com/datenstrom/yellow-plugins/tree/master/image)  
`[gallery photo.*jpg - 20%]` = add [image gallery](https://github.com/datenstrom/yellow-plugins/tree/master/gallery)  
`[youtube fhs55HEl-Gc]` = embed [Youtube video](https://github.com/datenstrom/yellow-plugins/tree/master/youtube)  
`[vimeo 13387502]` = embed [Vimeo video](https://github.com/datenstrom/yellow-plugins/tree/master/vimeo)  
`[soundcloud 101175715]` = embed [Soundcloud audio](https://github.com/datenstrom/yellow-plugins/tree/master/soundcloud)  
`[slideshare 37321279]` = embed [Slideshare presentation](https://github.com/datenstrom/yellow-plugins/tree/master/slideshare)  
`[googlemaps Stockholm]` = embed [Google maps](https://github.com/datenstrom/yellow-plugins/tree/master/googlemaps)  
`[blogrecent /blog/]` = show recent [blog pages](https://github.com/datenstrom/yellow-plugins/tree/master/blog)  
`[wikirecent /wiki/]` = show recent [wiki pages](https://github.com/datenstrom/yellow-plugins/tree/master/wiki)  
`[yellow]` = show [software version](https://github.com/datenstrom/yellow-plugins/tree/master/core)  

For a list of available shortcuts see [plugins](https://github.com/datenstrom/yellow-plugins).

## Settings

`Title` = page title  
`TitleContent` = page title shown in content  
`TitleNavigation` = page title shown in navigation  
`TitleHeader` = page title used in header  
`Description` = page description  
`Keywords` = page keyword(s), comma separated  
`Author` = page author(s), comma separated  
`Language` = page language  
`Theme` = page theme  
`Template` = page template  
`Modified` = page modification date, [YYYY-MM-DD](yellow-api#date-format) format  
`Published` = page publication date, [YYYY-MM-DD](yellow-api#date-format) format  
`Tag` = page tag(s) for categorisation, comma separated  
`Status` = page status for workflow  
`Navigation` = page navigation  
`Sidebar` = page sidebar  
`Redirect` = new location or URL  
`Robots` = robots meta data used in header  
`Sitename` = sitename used in header and footer  
`Parser` = parser for text files
